# Linting

## Обраний лінтер: HTMLHint

### Причини вибору
HTMLHint — простий та ефективний інструмент для перевірки HTML.

### Базові правила
- Нижній регістр тегів
- Подвійні лапки для атрибутів
- Всі теги закриті
- Обов’язкові alt для img

### Запуск
```
npx htmlhint index.html
```
